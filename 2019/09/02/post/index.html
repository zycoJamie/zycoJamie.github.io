支持es6的import/export

node.js
 2019-09-02 16:10:47  loading


nodejs本身支持大部分的es6语法，但当前任然不支持es6的import/export，因此借助babel

使得nodejs兼容es6的import/export语法

没有package.json的话就先npm init，在项目根目录创建 .babelrc文件

.babelrc

{
    "presets": [
        [
            "@babel/preset-env",
            {
                "targets":{
                    "node":true
                }
            }
        ]
    ],
    "plugins": [],
    
}

targets为node，能针对node环境，最小化的将不兼容代码转化为兼容代码（比如当前仅

仅是将es6的import/export转化为commonJs的require，而不是将所有es6代码转化为es5这

种全部转化）

npm install @babel/cli @babel/core @babel/node @babel/preset-env -D

最后通过npx执行本地的可执行程序，而不是使用 node 文件名 来执行

npx babel-node 文件名
