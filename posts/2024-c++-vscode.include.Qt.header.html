vscode #include Qt header 报错

Jul 20, 2024

#include QApplication
error: 无法打开源文件 "QApplication"
error: 检测到 #include 错误。请更新 includePath

c++
#include <QApplication>
#include "mainwindow.h"
 
int main(int argc, char *argv[])
{
    QApplication app(argc, argv);
 
    MainWindow mainWindow;
    mainWindow.show();
 
    return app.exec();
}

解决方案

  1. vscode打开command palette (command+shift+p)
  2. 选择c++:edit configuration(JSON)
  3. 生成c_cpp_properties.json文件
  4. 修改includePath，添加Qt安装路径(当前使用brew安装的Qt)
json
{
    "includePath": ["${workspaceFolder}/**",
    "/opt/homebrew/Cellar/qt/6.7.0_1/include/",
    "/opt/homebrew/Cellar/qt/6.7.0_1/lib/**"]
}
