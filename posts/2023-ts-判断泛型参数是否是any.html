判断泛型参数是否是any

Jun 01, 2023
ts
class Any {
    // @ts-expect-error – dont worry about this.
    private _: never
}
type B<T> = [T] extends [Any] ? number : string
let b:B<any> = 20

泛型参数使用' [ ] '包裹住，那么当泛型参数T是any时，满足extends，此时
type B = number

假如是如下定义

ts
type B<T> = T extends [Any] ? number : string
1

那么当泛型参数T是any时，此时
type B = number | string
