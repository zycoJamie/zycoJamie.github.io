web3项目npm安装依赖ethereumjs-abi失败

May 27, 2022

连接钱包的npm库 @walletconnect/web3-provider 依赖了 ethereumjs-abi，而依赖方式是

"ethereumjs-abi": "git+https://github.com/ethereumjs/ethereumjs-abi.git"

当使用npm install or yarn or pnpm i 安装依赖时，会去github上拉取ethereumjs-abi，因为是国内服务器，所以从github拉取依赖失败，项目打包失败

解决方法 ​

该项目包管理器使用的是pnpm，但服务器上打包，使用的是npm，因此本地运行

js
npm install --package-lock-only
1

仅生成package-lock.json文件，然后本地修改package-lock.json文件

修改前 ​
json
{
    "ethereumjs-abi": {
        "version": "git+https://github.com/ethereumjs/ethereumjs-abi.git#ee3994657fa7a427238e6ba92a84d0b529bbcde0",
        "from": "git+https://github.com/ethereumjs/ethereumjs-abi.git"
    }
}

json
{
    "ethereumjs-abi": {
        "version": "0.6.8",
        "resolved": "https://registry.npmjs.org/ethereumjs-abi/-/ethereumjs-abi-0.6.8.tgz"
    }

服务器上的打包脚本 使用npm ci

ethereumjs-abi成功install，项目打包成功
